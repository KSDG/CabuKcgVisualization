高雄市戶政資料視覺化

專案緣起

因高雄市發生氣爆，KSDG、Cocoaheads Kaohsiung、Code For Kaohsiung 共同舉辦 寫Code救家園聯合駭客松，初次聚會討論以戶政資料為基礎建設，此為戶政資料視覺化之樣板。

first project

初次討論發生「聽說」用高雄市出生與死亡率做為人口老年化資料佐證，如此有邏輯問題之結論需要被檢視，所以當場用高雄市民政局人口資料做出以下視覺分析，相關活動討論請參考討論1、討論2。

成果網站化

原始圖片僅為利用 Excel 產生之視覺分析結果，對於分享與發佈效果不彰，利用 C3.js 函式庫建立易於分享之網頁內容。

資料來源

主要資料來自於 高雄市民政局公開資料，初始資料來自於更多 首頁 => 人口統計 => 高雄市戶籍人口統計查詢 => 出生與死亡統計，主要取材於 這頁
